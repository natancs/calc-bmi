# BMI Calculator

Project in React + TypeScript creating a BMI calculator.

Project developed during the [B7Web](https://b7web.com.br) course.

### Installation
- `npm install` or `yarn`

### To rotate
- `npm run start` or `yarn start`

## Applied concepts
- Componentization
- CSS Modules
- Reack Hooks (_*useState*_( ))
- Tipagens com o TypeScript
- Responsiveness
